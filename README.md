# smart-fall-detection-system

## Overview
An ESP32-based wearable system for detecting falls and recognizing human activities using MPU6050 sensor data and BLE communication.

## Features
- Real-time activity recognition
- Fall detection
- BLE communication
- Sensor monitoring

## Hardware Components
- ESP32
- MPU6050
- MQ135
- PIR Sensor

## Planned Features
- ML-based activity classification
- Mobile application
- Cloud dashboard
- Emergency alert system

## Folder Structure

firmware/ -> ESP32 source code
datasets/ -> collected sensor data
ml-models/ -> machine learning models
docs/ -> documentation and diagrams
hardware/ -> circuit diagrams and wiring
tests/ -> testing scripts
